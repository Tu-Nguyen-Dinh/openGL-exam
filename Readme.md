# openGL-exam
## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/Tu-Nguyen-Dinh/openGL-exam.git
~~~

Go to the project directory  

~~~bash  
  cd openGL-exam
~~~

Install dependencies  
~~~bash 
  sudo apt-get install libglut3-dev
~~~

Start 

~~~bash  
g++ cube.cpp -o cube -lGL -lGLU -lglut
./cube
~~~
